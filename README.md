## How to Fork This Repository

If you would like to use this repository for your own study or reference:

1. **Fork the repository:**
   - Click the "Fork" button at the top right corner of this page. This will create a personal copy of the repository under your own GitHub account.

2. **Clone your fork:**
   - Visit your forked repository (it will be under your GitHub username).
   - Click the "Code" button and copy the URL.
   - Open your terminal and run:
     ```
     git clone https://github.com/YOUR-USERNAME/bcs322.git
     ```
   - Replace `YOUR-USERNAME` with your actual GitHub username.

3. **Study and work locally:**
   - You can now review and modify the code on your computer.
   - This allows you to experiment with changes without affecting the original repository.

4. **Sync with updates:**
   - If you want to keep your fork up-to-date with this repository, you can add an upstream remote:
     ```
     git remote add upstream https://github.com/trtlbby/bcs322.git
     git fetch upstream
     git merge upstream/main
     ```
   - This will pull in changes if this repository is updated.
